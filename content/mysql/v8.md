使用docker运行mysql 8
docker run -p 3306:3306 --name mysql -e MYSQL_ROOT_PASSWORD=密码 -d mysql:8
由于密码加密方式与以前不同，所以需要进入容器更改密码加密方式
docker exec -it mysql /bin/bash
ALTER USER 'root'@'%' IDENTIFIED WITH mysql_native_password BY '密码';
